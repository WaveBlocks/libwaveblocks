Detailed Description

template<dim_t D, class S>
class waveblocks::ShapeExtensionEnumeration< D, S >

Extended shapes are needed for the gradient computation of the wavepacket.

This class is not used anymore. The original idea was that the enumeration of the extended shape consists of two parts: The first part is equal to the enumeration of the original shape. This class forms the second part. It stores the additional nodes that are part of the shape extension 'shell'. This saves some memory as the first part is identical to the enumeration of the corresponding basic shape. Since the gradient computation involves the evaluation of a wavepacket, this two-part idea implicates that two implementations of the evaluation are needed. One implementation for a wavepacket using a basic shape and one for using an extended shape. This two-part idea has been abandoned for the sake of simplicity.

This class has been fully tested and works.
