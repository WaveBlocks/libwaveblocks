Detailed Description

template<class UINT, dim_t D>
class waveblocks::TinyMultiIndex< UINT, D >

Represents the whole multi-index using a single integer.

This implementation splits an integer into same sized parts. For example using a 64bit integer for 20 entries yields only 3 bits per entry. This means when using this class, special care must be taken to prevent overflows. When running in release mode, input is not checked.

Template Parameters
UINToption to select which integer type to use
Dnumber of multi-index dimension i.e number of entries
