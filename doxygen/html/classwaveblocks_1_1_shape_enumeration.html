Detailed Description

template<dim_t D>
class waveblocks::ShapeEnumeration< D >

Assigns all multi-indices of a shape an ordinal.

Since many algorithms operating on wavepackets use recursive formulas, shape enumerations are divided into a set of slices to simplify data structures.

The \( s \)-th slice contains all multi-indices \( \boldsymbol{k} \in \mathfrak{K} \) that satisfy \( \displaystyle\sum_{d=1}^{D} k_d = s \).

To determine, to which slice a multi-index belongs, do:

#include <algorithms>
int islice = std::accumulate(index.begin(), index.end(), int(0));
