Detailed Description

template<dim_t D, class MultiIndex>
class waveblocks::DefaultShapeSlice< D, MultiIndex >

Default implementation of a shape enumeration.

This class takes a shape description object and builds a lookup-table to perform queries. It lets the user freely choose an appropriate type to represent multi-indices internally.

Implementation details

This class uses a vector to do ordinal -> multi-index queries. For multi-index -> ordinal it does binary search by default. The constructor provides an option to use an hashmap instead. However it turned out that binary search is slightly faster than a hashmap regardless of the hash-function.

Template Parameters
Dnumber of multi-index dimensions
MultiIndexType to internally represent a multi-index.
A custom type should provide same interface as std::array<int,D>.
Furthermore a custom type must specialize std::less, std::hash, std::equal_to.
Sshape description class
See also
TinyMultiIndex A compressed multi-index type that represents all multi-indices using a single integer.

Returns the position of the node with multi-index index.

Notice that the first node in the slice has position 0 (not 1 or offset()).

Portable programs should never call this function with an node that is not part of this slice since this causes undefined behaviour.

Use ShapeEnumeration<D>::contains(index) to check whether this slice contains the given node.

complexity: logarithmic in the number of slice-nodes

inlineoverridevirtual

Retrieves ordinals of all backward neighbours of a given node very efficiently.

Notice that this method assumes that the given node is part of the shape. Therefore this method does not need to perform any contains()-checks since it knows that all backward neighbours exist, except when the given node contains some zero entries. In the latter case, this method returns an undefined ordinal.

If the given node is not part of the shape, then the behaviour is undefined.

For each backward neighbour its ordinal. An ordinal is undefined if its node does not exist.
