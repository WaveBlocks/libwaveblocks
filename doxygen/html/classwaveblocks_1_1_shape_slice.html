template<dim_t D>
virtual std::size_t waveblocks::ShapeSlice< D >::find ( const std::array< int, D > &  index) const
pure virtual

Returns the position of the node with multi-index index.

Notice that the first node in the slice has position 0 (not 1 or offset()).

Portable programs should never call this function with an node that is not part of this slice since this causes undefined behaviour.

Use ShapeEnumeration<D>::contains(index) to check whether this slice contains the given node.

complexity: logarithmic in the number of slice-nodes

template<dim_t D>
virtual std::array<std::size_t,D> waveblocks::ShapeSlice< D >::find_backward_neighbours ( const std::array< int, D > &  index) const
pure virtual

Retrieves ordinals of all backward neighbours of a given node very efficiently.

Notice that this method assumes that the given node is part of the shape. Therefore this method does not need to perform any contains()-checks since it knows that all backward neighbours exist, except when the given node contains some zero entries. In the latter case, this method returns an undefined ordinal.

If the given node is not part of the shape, then the behaviour is undefined.

Parameters
[in]indexnode that is part of the shape
Returns
For each backward neighbour its ordinal. An ordinal is undefined if its node does not exist.
